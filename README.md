# SERaaS' API Service
Outputs emotion statistics from audio files containing speech, as an API.

## General

### Technologies Used

#### Python
* LibROSA (Audio Feature Extraction)
* SciKit-Learn (Machine Learning Model)
* joblib (Model Persistence)

#### JavaScript
* Node.js
* Swagger.js

#### Other
* ffmpeg (File Splitting)
